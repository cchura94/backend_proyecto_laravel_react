# 3. Comandos iniciales
- Si el repositorio ya existe
```
git clone <direccion>
```
- Si no existe el repositorio
```
git init
```
- Vincular nuestro repositorio local con el repositorio remoto
```
git remote add origin direccion_de_github
```

# 4. Comandos de GIT

```

git push origin master
```
# 5. Actualizar
